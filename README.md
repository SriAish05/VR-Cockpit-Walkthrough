# ✈️ VR Cockpit Walkthrough — Boeing 737-800

An interactive first-person VR training application 
that places the user inside a photorealistic Boeing 
737-800 cockpit.

## 🎮 Features
- First-person 360° cockpit exploration
- Interactive instrument labels (Yoke, Thrust Levers)
- Aviation HUD aesthetic (cyan text, dark panels)
- Cockpit ambient audio
- Meta Quest 3 + PC support via OpenXR
- Mouse-look navigation

## 🛠️ Tech Stack
- Unity 2022.3 LTS
- C# Scripting
- OpenXR / Meta Quest 3
- TextMeshPro
- AR Foundation
